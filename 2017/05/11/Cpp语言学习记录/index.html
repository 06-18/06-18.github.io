语言知识点

unsigned

尽量不要用unsigned int 变量进行运算。因为与 unsigned 一起运算，int会转换成无符号数。并且结果也是无符号数。当相减得到负数时，会取模。这肯定不是你想要的

char , signed char , unsigned char是三种不同的字符型，不过在表现形式上是两种（有符号，无符号）

原因是。因为char在一些机器上是有符号的，而在另一些机器上又是无符号的，所以如果使用char进行运算特别容易出问题。如果你使用char来表示一个不大的整数，那么必须明确指定它的类型是signed char 或者 unsigned char。

列表初始化

初始化形式有以下几种

函数内的内置类型变量，需要显示初始化

在函数体内的需要显示初始化int a = 0, 而位于函数体外的内置类型，默认已经初始化为0

const（常量）

在编译成.o时，就使用该常量的地方会被替换成对应的常量值。而不在是去读取变量。

constexpr

对指针来说，只是表明是指针常量。其它变量和函数则会再加 const在

typedef 和 using

顶层const 和 底层const

变量本身的const就是顶层const， 而指针变量所指向的变量的const就是底层const

auto会忽略顶层const，保留底层const

decltype

获取表达式或者函数的类型，但不会用该表达式或者函数初始化变量；

与auto的不同点

decltype不会忽略顶层const
