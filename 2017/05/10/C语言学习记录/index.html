变量声明和定义的区别

extern int a; //这里是外部变量的声明
int a;//函数外，也就是说这个是一个外部变量。这里是定义

初始化

在不显示的初始化情况下，静态变量和外部变量都将被初始化为0，而自动变量（局部变量）和寄存器变量的初值则没有定义。

char

  1. 与java中的btye类似，都是一个字节长度。
  2. unsigned char 范围 0~255 signed char 范围 -128~127

EOF == -1

const

  1. int strlen(const char[])不能修改数组中的元素

char a[] 与 char* a 区别

  1. a[]是数组，是变量，可以改变； char* a 是字符串常量，不可修改

  2. 作为函数局部变量时的区别

    返回的是指向常量的指针，该常量在编译的时候就已经分配好了。所以不会在离开函数作用域之后被销毁。这样使用没有问题

返回的是指向局部变量的指针，局部变量在离开局部作用域之后会被销毁，所以返回的是指向被销毁的变量地址。这样使用是错误的
